# PyRanker
Rank agreggation library in Python. PyRanker have implemented 12 different rank aggregation algorithms.

# Rank Agregation Algorithms
PyRanker have the following algorithms implemented:
- FootRule
- Borda
- Copeland
- Median Rank
- MScores
- Reciprocal
- CombANZ
- CombMAX
- CombMIN
- CombMED
- CombMNZ
- CombSUM
